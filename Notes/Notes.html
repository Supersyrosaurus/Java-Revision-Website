Java Revision Notes

1. Introduction to Java

Java is a high-level, object-oriented programming language developed by Sun Microsystems (now owned by Oracle).

It was released in 1995 and is widely used for various types of applications.

Java is platform-independent due to the Java Virtual Machine (JVM), making it Write Once, Run Anywhere (WORA).

It supports multithreading, garbage collection, and a large standard library.

2. Basic Syntax

Every Java program starts with a class.

The main method is the entry point:

public class Main {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}

Java is case-sensitive, and statements end with a semicolon (;).

3. Data Types & Variables

Primitive Data Types: int, double, char, boolean, float, long, byte, short

Reference Data Types: Objects, Arrays, Strings

int age = 25;
double price = 19.99;
char grade = 'A';
boolean isJavaFun = true;

4. Operators

Arithmetic Operators: +, -, *, /, %

Relational Operators: ==, !=, >, <, >=, <=

Logical Operators: &&, ||, !

Assignment Operators: =, +=, -=, *=, /=

5. Control Flow Statements

If-Else Statement:

if (age > 18) {
    System.out.println("Adult");
} else {
    System.out.println("Minor");
}

Loops:

For Loop:

for (int i = 0; i < 5; i++) {
    System.out.println(i);
}

While Loop:

int i = 0;
while (i < 5) {
    System.out.println(i);
    i++;
}

6. Methods & Functions

A method is a block of code that performs a specific task.

public static int add(int a, int b) {
    return a + b;
}

Methods can have parameters and return types, and they help in code reusability.

7. Object-Oriented Programming (OOP)

Classes & Objects

class Car {
    String model;
    int year;
    Car(String model, int year) {
        this.model = model;
        this.year = year;
    }
}

Encapsulation: Hiding data using private keyword

Inheritance: Using extends to inherit a class

Polymorphism: Method Overloading & Overriding

Abstraction: Hiding implementation using abstract classes and interfaces
